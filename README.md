# Shop Hub - Modern E-Commerce Platform

A sleek, responsive e-commerce frontend application built with React, featuring advanced caching, real-time stock management, and seamless user experience.

## Features

- **Product Catalog**: Browse 40+ products with detailed information
- **Smart Search**: Real-time search with suggestions and category filtering
- **Shopping Cart**: Persistent cart with stock validation
- **Checkout Process**: Complete checkout simulation with order confirmation
- **Order History**: Track and view past orders
- **User Authentication**: Login system with profile management
- **Wishlist**: Save favorite products for later
- **Stock Management**: Real-time inventory tracking
- **Responsive Design**: Perfect on all devices

## Tech Stack

- **React 18** - Modern React with hooks
- **Vite** - Fast development and builds
- **React Router** - Client-side navigation
- **Context API** - Global state management
- **CSS Variables** - Dynamic theming
- **localStorage** - Data persistence
